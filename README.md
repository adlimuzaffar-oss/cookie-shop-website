# Cookie Shop Website

Welcome to the Cookie Shop Website project! This project is designed to showcase and sell a variety of delicious cookies through an engaging and user-friendly web interface.

## Project Structure

The project is organized as follows:

```
cookie-shop-website
├── src
│   ├── index.html          # Main HTML file for the website
│   ├── css
│   │   └── styles.css      # Custom styles for the website
│   ├── js
│   │   └── main.js         # JavaScript functionality for interactivity
│   ├── components
│   │   └── cookie-card.html # HTML structure for the Bootstrap card component
│   └── assets
│       ├── fonts           # Directory for custom font files
│       └── icons           # Directory for icon files
├── .gitignore              # Specifies files to be ignored by version control
├── package.json            # Configuration file for npm
└── README.md               # Documentation for the project
```

## Features

- **Responsive Design**: The website is built using Bootstrap to ensure it looks great on all devices.
- **Navbar**: Easy navigation to different sections of the website.
- **Hero Section**: A visually appealing banner to welcome visitors.
- **Brand Introduction**: A section that introduces the cookie brand and its values.
- **Product Showcase**: Bootstrap card components to display individual cookie products.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd cookie-shop-website
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Open `src/index.html` in your web browser to view the website.
